# Boso Suroboyoan — cooking translation guide

The household cook is from **Surabaya, East Java**, and reads **Boso Suroboyoan** (the Surabaya /
"Arekan" dialect of Javanese). Translate each recipe's ingredient names, notes, and steps into this
dialect for the `_su` fields.

## Register & style
- Write **Ngoko** (everyday informal Javanese) — the blunt, egalitarian Surabaya register, the way a
  cook talks in the kitchen. **Not** formal/standard Indonesian, and not high Javanese (Krama).
- Keep it **clear and practical** over flowery. Short imperative steps.
- **Numbers and metric units never change** — only the words translate. `270 ml` → `270 ml`,
  `180 °C` → `180 °C`, `10–12 menit`.
- Code-mixed kitchen loanwords that Surabayans actually use are fine (`kecap`, `mentega`, `porsi`,
  `saus`, `oven`). Don't force an obscure Javanese word when a common one is universally understood.
- This guide is a starting point — if the cook corrects a word, prefer their version next time.

## Dialect markers (what makes it Suroboyoan, not standard)

|--------------------|-------------------|-----------------------|

| at / to / in       | **nang**, ndhik   | ana ing / di          |
| with / and         | **ambek**, ambi   | karo / dengan         |
| this / that        | **iki / iku**     | ini / itu             |
| the aforementioned | **mau**           | tadi                  |
| so that            | **ben**           | supaya / agar         |
| until              | **ngasi**, nganti | sampai                |
| using / wear       | **nganggo**       | nganggo / pakai       |

## Handy step phrases
- low / medium / high heat → **geni cilik / geni sedheng / geni gedhe**
- until cooked / until boiling → **ngasi mateng / ngasi umob**
- stir evenly → **aduk ngasi rata**
- set aside → **disèlèhno disik**
- serve warm / hot → **sajekno anget-anget / panas-panas**
- optional / not required → **gak wajib**
- room temperature → **suhu kamar**
- dark in **colour** (e.g. strong tea, soy) → **ireng** (NOT `peteng`, which means dark = no light)
- surface / top face → **lumah / lumahane** (prefer over the Indonesian `permukaan`)

## Worked example
English ingredient: `{ "item": "Garlic", "note": "minced" }`
→ `"item_su": "Bawang putih", "note_su": "dicacah lembut"`

English step: "Melt the butter over medium heat and fry the garlic until fragrant."
→ `"text_su": "Lelehno mentega nganggo geni sedheng, terus goreng bawang putih ngasi mambu wangi."`
